# Posts App

A vanilla HTML, CSS, and JavaScript application that fetches posts from an API and allows users to favorite and unfavorite posts.

## Features

- Fetches posts from an API
- Displays 10 posts
- Favorite/unfavorite features
- Favorites stored in localStorage

## Planned Features

- Search through post content
- Highlight matching search terms
- Additional UI improvements

## Technologies

- HTML
- CSS
- JavaScript
